# Functional Addition

[Codewars Kata Link](https://www.codewars.com/kata/538835ae443aae6e03000547/python)

## Description
Create a function add(n)/Add(n) which returns a function that always adds n to any number

```python
add_one = add(1)
add_one(3)  # 4

add_three = add(3)
add_three(3) # 6
```

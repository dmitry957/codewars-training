# Make a square box!

[Source](https://www.codewars.com/kata/58644e8ddf95f81a38001d8d/python)

## Description
Easy; Make a box
Given a number as a parameter (between 2 and 30), return an array containing strings which form a box.
Like this:

n = 5

[
  '-----',
  '-   -',
  '-   -',
  '-   -',
  '-----'
]
n = 3

[
  '---',
  '- -',
  '---'
]
